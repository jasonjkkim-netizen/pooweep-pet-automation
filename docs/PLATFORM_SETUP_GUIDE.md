# 🐾 Pooweep 펫 쇼핑몰 플랫폼별 가입 및 설정 가이드

## 📌 현재 상태 (2025.03)
- **스토어**: pooweep.co.kr (Cafe24)
- - **상품수**: 47개 (드랍쉬핑 + 자체상품)
  - - **카테고리**: 사료/간식, 용품, 외출/이동, 패션, 건강/위생
   
    - ---

    ## 1️⃣ 사방넷 (Sabangnet) - 멀티채널 통합관리

    ### 가입 순서
    1. https://www.sabangnet.co.kr 접속
    2. 2. 무료체험 신청 클릭
       3. 3. 사업자등록번호: 3592701761 입력
          4. 4. 상호명: 베리스 / 대표자: 김종국
             5. 5. Cafe24 연동 설정: API 키 입력
                6. 6. 판매채널 추가: 네이버 스마트스토어, 쿠팡, 11번가 등
                  
                   7. ### Cafe24 자동 연동 방법
                   8. - Cafe24 관리자 > 마켓플러스 메뉴에서 직접 연동 가능
                      - - 경로: pooweep.cafe24.com > 마켓플러스 > 마켓 연동
                       
                        - ### 월 비용
                        - - 기본: 33,000원/월
                          - - 주문 1,000건 이상: 55,000원/월
                           
                            - ---

                            ## 2️⃣ 네이버 스마트스토어 (Naver SmartStore)

                            ### 가입 순서
                            1. https://sell.smartstore.naver.com 접속
                            2. 2. 네이버 계정으로 로그인
                               3. 3. 판매자 센터 가입 신청
                                  4. 4. 사업자정보 입력 (사업자등록번호: 3592701761)
                                     5. 5. 정산 계좌 등록
                                        6. 6. 스토어 정보 설정 (스토어명, 로고, 카테고리)
                                           7. 7. 상품 등록 시작
                                             
                                              8. ### Cafe24에서 자동 연동
                                              9. - Cafe24 관리자 > 마켓플러스 > 네이버 쇼핑 연동
                                                 - - 또는 사방넷을 통해 일괄 상품 등록
                                                  
                                                   - ### 수수료
                                                   - - 카테고리별 5.5~13% (반려동물용품: 약 8~10%)
                                                    
                                                     - ---

                                                     ## 3️⃣ 쿠팡 (Coupang) 마켓플레이스

                                                     ### 가입 순서
                                                     1. https://wing.coupang.com 접속
                                                     2. 2. 판매자 가입 신청
                                                     3. 사업자등록증 업로드
                                                     4. 4. 통장사본 업로드
                                                        5. 5. 판매자 인증 완료 (1~3일 소요)
                                                           6. 6. WING 판매자센터에서 상품 등록
                                                             
                                                              7. ### Cafe24에서 자동 연동
                                                              8. - Cafe24 관리자 > 마켓플러스 > 쿠팡 연동
                                                                 - - 쿠팡 WING API 키 발급 후 사방넷에서 연동
                                                                  
                                                                   - ### 수수료
                                                                   - - 카테고리별 7~15% (반려동물: 약 10.8%)
                                                                    
                                                                     - ---

                                                                     ## 4️⃣ 네이버 광고 (Naver Search Ads)

                                                                     ### 가입 순서
                                                                     1. https://searchad.naver.com 접속
                                                                     2. 2. 네이버 비즈니스 계정으로 로그인
                                                                        3. 3. 광고 계정 생성
                                                                           4. 4. 결제수단 등록 (신용카드/계좌이체)
                                                                              5. 5. 키워드 광고 캠페인 생성
                                                                                 6. 6. 추천 키워드: 강아지간식, 반려동물용품, 강아지사료, 펫용품
                                                                                   
                                                                                    7. ### 자동화 방법
                                                                                    8. - 네이버 검색광고 API 활용
                                                                                       - - 자동입찰, 키워드 성과 분석 스크립트 (scripts/naver_ads_manager.py)
                                                                                        
                                                                                         - ### 예산 가이드
                                                                                         - - 시작: 일 10,000~30,000원
                                                                                           - - 운영 안정 후: 일 50,000~100,000원

                                                                                             ## 5️⃣ 카카오모먼트 (Kakao Moment)

                                                                                             ### 가입 순서

                                                                                             2. 2. 카카오계정 로그인
                                                                                                3. 3. 광고 계정 생성
                                                                                                   4. 4. 비즈니스 정보 입력
                                                                                                      5. 5. 결제수단 등록
                                                                                                         6. 6. 카카오톡 채널 연동 (pooweep 채널 생성 필요)
                                                                                                            7. 7. 디스플레이 광고 또는 메시지 광고 캠페인 생성
                                                                                                              
                                                                                                               8. ### 추천 광고 유형
                                                                                                               9. - 카카오톡 비즈보드: 메인 피드 노출
                                                                                                                  - - 카카오스토리 광고: 반려동물 커뮤니티 타겟
                                                                                                                    - - 카카오톡 채널 메시지: 구매고객 리마케팅
                                                                                                                     
                                                                                                                      - ### 예산 가이드
                                                                                                                      - - 시작: 일 5,000~20,000원

                                                                                                                        ## 6️⃣ 채널톡 (Channel.io) - CRM 자동화
                                                                                                                        
                                                                                                                        ### 가입 순서
                                                                                                                        1. https://channel.io 접속
                                                                                                                        2. 2. 무료 플랜으로 시작
                                                                                                                           3. 3. 채널톡 스크립트를 Cafe24에 설치
                                                                                                                              4. 4. Cafe24 관리자 > 앱스토어 > 채널톡 검색 > 설치
                                                                                                                                
                                                                                                                                 5. ### Cafe24에서 자동 설치 방법
                                                                                                                                 6. - Cafe24 관리자 > 앱 > 앱스토어 > "채널톡" 검색
                                                                                                                                    - - 원클릭 설치 가능 (별도 코드 삽입 불필요)
                                                                                                                                     
                                                                                                                                      - ### 자동화 기능
                                                                                                                                      - - 자동 환영 메시지
                                                                                                                                        - - 주문 상태 자동 알림
                                                                                                                                          - - FAQ 자동응답 봇
                                                                                                                                            - - 고객 세그먼트별 마케팅 메시지

                                                                                                                                              ## 7️⃣ Cafe24 마켓플러스 (내장 기능으로 한번에!)
                                                                                                                                              
                                                                                                                                              ### ⭐ Cafe24에서 자동으로 한번에 할 수 있는 방법
                                                                                                                                              
                                                                                                                                              Cafe24 마켓플러스를 사용하면 별도 사방넷 가입 없이도 멀티채널 판매가 가능합니다.
                                                                                                                                              
                                                                                                                                              ### 설정 방법
                                                                                                                                              1. Cafe24 관리자 로그인
                                                                                                                                              2. 2. 좌측 메뉴 > 마켓플러스 클릭
                                                                                                                                                 3. 3. 연동 가능 마켓 확인:
                                                                                                                                                    4.    - 네이버 쇼핑
                                                                                                                                                          -    - 쿠팡
                                                                                                                                                               -    - 11번가
                                                                                                                                                                    -    - G마켓/옥션
                                                                                                                                                                         -    - 카카오톡 스토어
                                                                                                                                                                              -    - 인터파크
                                                                                                                                                                                   - 4. 각 마켓별 판매자 계정 연동
                                                                                                                                                                                     5. 5. 상품 일괄 등록/수정
                                                                                                                                                                                        6. 6. 주문 통합 관리
                                                                                                                                                                                           7. 7. 재고 자동 동기화
                                                                                                                                                                                             
                                                                                                                                                                                              8. ### 비용
                                                                                                                                                                                              9. - Cafe24 마켓플러스: 월 11,000원~
                                                                                                                                                                                                 - - 각 마켓 수수료는 별도
                                                                                                                                                                                                  
                                                                                                                                                                                                   - ### 장점
                                                                                                                                                                                                   - - Cafe24 관리자에서 모든 채널 통합 관리
                                                                                                                                                                                                     - - 재고 자동 동기화
                                                                                                                                                                                                       - - 주문 통합 수집
                                                                                                                                                                                                         - - 별도 외부 솔루션 불필요
